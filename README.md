# Planvas Backend
🛠 Tech Stack Summary
Backend Core
Runtime: Node.js (v18+ or v20+)

Framework: Express.js

Language: JavaScript / Node.js

ORM: Prisma (Type-safe Database Client)

Authentication: JWT (JSON Web Token) based OAuth 2.0

Database
RDBMS: MySQL (8.0+)

Database Design: Relational Schema (User, Goal, Task, etc.)

Infrastructure & DevOps
Cloud Hosting: AWS EC2 (Ubuntu 22.04 LTS)

Process Manager: PM2 (Zero-downtime & Environment Management)

Web Server: Nginx (Reverse Proxy & SSL/TLS Termination - Planned)

DNS: AWS Route 53 (Planned)

Tools & API Documentation
API Spec: Swagger (OpenAPI 3.0)

Database Tool: MySQL Workbench / Prisma Studio

Version Control: Git & GitHub
